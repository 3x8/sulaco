# Board - HAKRCF411


### Hardware and Features

  - STM32CubeMX 	
  - MCU	STM32F411CEUx
  - IMU: MPU6000 (SPI) 
  - VCP: yes
  - OSD: Betaflight OSD
  - Battery Voltage Sensor: yes
  - Stable voltage regulation,9V/2A DCDC BEC for VTX/camera etc.And could select 5v/9v with pad
  - Serial LED interface(LED_STRIP)
  - 2 UART 
  - BARO: QMP6988/BMP280(I2C_1)
  - GPS
  -
